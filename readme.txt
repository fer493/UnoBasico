Juego UNO (Consola - Java)
Descripción:
    Implementación del juego UNO en consola con soporte para múltiples jugadores, inteligencia artificial básica y reglas principales del juego.

Características:
    1 jugador humano + 3 jugadores IA
    Reglas básicas del UNO:
    Salto
    Reversa
    Roba 2
    Roba 4
    Comodín
    Sistema de turnos dinámico
    Validación de entradas
    Manejo de errores

Tecnologías:
    Java
    Programación orientada a objetos

Ejecución
Compilar y ejecutar:
    javac *.java
    java Game
